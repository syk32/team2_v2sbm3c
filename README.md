# 🐥 우리들의 추억저장소 웹 팀 프로젝트

**아이와 AI의 대화를 통해 하루 일과를 일기로 자동 생성하는 웹 서비스**

- 📆 **개발 기간** : 2024.12 ~ 2025.01

---

## ✨ 주요 기능

- 🔐 사용자 로그인 및 개인별 데이터 관리
- 📋 설문조사를 통한 아이 상태 및 감정 정보 수집
- 🗂️ 사용자 소통을 위한 게시판 기능
- 🤖 AI 챗봇과의 대화를 통한 하루 기록
- 🗓️ 대화 기반 텍스트·그림 일기 자동 생성
- 💟 좋아요 기능

---

## ⛏ 기술 스택 


### Frontend 
- HTML / CSS : 전체 화면 구조 및 레이아웃 구성 
- JavaScript : AI 챗봇 대화 처리 및 사용자 인터랙션 구현 
- Thymeleaf : 서버 데이터와 연동되는 동적 화면 렌더링

### Backend 
- Spring Boot : 로그인, 게시판, 설문조사, AI 일기 생성 API 구현 
- MyBatis : SQL 기반 데이터 접근 및 Mapper를 통한 DB 연동 
- REST API : AI 챗봇 대화, 일기 생성 요청·응답 처리 
- Oracle Database : 사용자, 일기, 설문 등 데이터 저장
